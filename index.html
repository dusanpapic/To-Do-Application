Hello there!
This is the guide that will help you use this app (if you already don't know how to use it).
To add new items to the list, you can use Type To Add... input field.
After typing in desired text, Plus button will add it to the list underneath.

Next to every item on the list, there are two buttons, Trash button and Check button.
Trash button will remove selected item from the list.
Check button completes selected task and moves it to the bottom of the list.
Star option on the left side allows user to mark selected task.

If you want to search for concrete item, you can use Search input field. Just start typing and find desired item.
Filter options on the right side allows you to hide completed or uncompleted items.

On the upper right corner, you can see three buttons. Light button will turn on the lights (Light theme).
Reset button is there to help delete everything and start over (don't use it if you are not done with your tasks).
You already know what third button does, so I'm not going to bother explaining it.

There is nothing more to say than thanks for visiting and I hope you like my app.
See ya!
